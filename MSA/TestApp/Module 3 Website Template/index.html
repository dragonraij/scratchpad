This app is based around the material covered in the first 20 videos of the Windows Store apps for Absolute Beginners with C# video series on the Channel 9 site. The lessons were used to help create a custom app.

Things that have been learnt and applied in the app are app design, working with XML, C# programming, working with data files, using json in apps, data binding, use of the grid app layout.
